A Muttator Reference Sheet / Cheat Sheet

Muttator v0.6 for Thunderbird v3.0 and 3.1.* (muttator-20100629.xpi)

The main thing to know about using Muttator is that there are two primary modes, EX mode and MESSAGE mode, and keyboard mappings can work very differently in the two modes. It's not obvious which mode you're in; you have to look at the bottom left of your Thunderbird window where the status line with either be blank (EX mode) or will say "-- MESSAGE --". There is also an odd "-- CARET --" mode that you'll probably want to out of as soon as possible.

When composing a new message, the two primary modes are "-- COMPOSE --" and "-- INSERT --". While typing in INSERT mode, you can press at any time to return to COMPOSE mode. Mappings in EX mode

i "Switch to MESSAGE mode" , I "Open the message in new tab" "Scroll message or select next unread one" t "Select thread" d, "Move mail to Trash folder" s "Move selected messages" S "Copy selected messages" u "Undo" "Redo" gg "Select first message" G "Select last message" + "Scroll message down pagewise" - "Scroll message up pagewise" j, "Select next message" gj "Select next message, including closed threads" J, "Select next unread message" k, "Select previous message" gk "Select previous message" K "Select previous unread message" * "Select next message from the same sender"

"Switch to EX mode" f "Follow hyperlinks" / "Search forward for a pattern" ?
